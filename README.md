# 🏠 Saudi Arabia Real Estate Analysis

A comprehensive data analysis project examining **1,480 real estate properties** across 4 major Saudi cities to uncover pricing patterns and market insights.

## 📊 Project Overview

This project analyzes real estate data from Riyadh, Jeddah, Dammam, and Khobar to answer key questions about property pricing, location value, and the impact of amenities.

## 🎯 Key Questions Answered

- Which city has the most expensive properties?
- What factors most influence property prices?
- How do amenities (pool, garage, basement) affect pricing?
- Which districts offer the best value?

## 🔍 Key Findings

- **Jeddah** is the most expensive city (avg. 113,778 SAR)
- **Bedrooms have almost no impact** on price (correlation: 0.01)
- **Basement** increases price the most (+78.6%)
- Properties aged **6-10 years** are the most expensive
